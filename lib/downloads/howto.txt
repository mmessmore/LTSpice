# How to make an opamp

1. Open one of these files
2. Click on the name and tell it to be a symbol
3. Save and close
4. Open lib/sym/AutoGenerated/[name].asy
5. Open lib/sym/OpAmp/UniversalOpAmp2.asy
6. Replace everything except SYMATTR from [name].asy with the conents except SYMATTR from UniversalOpAmp2.asy
7. Open lib/sym/AutoGenerated/[name].asy in LTSpice
8. Add a text label for the model and save
